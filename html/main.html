SHA256 with RSA

Sign
require private key
Sign your message, so the recipient can trust that the message is sent from you.
Verify
require public key, signature
When you hold the public key and receive a signed message, this application verifies if the signature of the message is valid.
Generate
Generate a new RSA keypair.
Encrypt
require public key
When you hold the public key, this application encrypts the message, so only the holder of the private key can decrypt your message.
Decrypt
require private key
When you hold the private key and a message which is encrypted by your public key, this application decrypt the message.
